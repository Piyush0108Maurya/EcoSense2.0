# EcoSense — Environmental Intelligence Platform
Developed with React + Vite + Gemini AI + WAQI

## 🚀 How to Start
1. Ensure Node.js is installed.
2. Run:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` in your browser.

## 📁 Project Structure (React)
- `/src/App.jsx`: State controller and view router.
- `/src/components/common/Navbar.jsx`: Smart navigation (Tabs & Profile).
- `/src/components/Landing/Landing.jsx`: Animated marketing section.
- `/src/services/api.js`: AI (Gemini) and Air Quality (WAQI) integration.
- `/src/services/db.js`: LocalStorage gamification logic.
